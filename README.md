## 🎮 Overview

Star Catcher VR is a colorful mobile VR game built using A-Frame.
The goal is to catch glowing stars by gazing at them while avoiding obstacles.


## 🚀 How to Run

### Option 1 – Play Online

* Open the GitHub Pages link
* Use Google Chrome
* Tap the VR button

### Option 2 – Run Locally

* Download project
* Run:

```
python3 -m http.server 8080
```

* Open:

```
http://localhost:8080
```

## ⚙️ Features

* Flash screen
* Setup menu
* Multiple levels
* Sound effects + music
* Scoreboard
* Gaze controls
* VR mode
* Custom textures
* Collision detection



## 🕹️ Controls

* VR: Look and hold gaze
* Desktop: Mouse + click
* Mobile: Tap and drag


## 🎯 How to Play

* Catch stars ⭐
* Avoid rocks 🪨
* Score as high as possible
